token.js ¬∑ Quick Start

token.js lets you create and validate simple tokens on the client-side. No server required ‚Äî it uses localStorage to remember which tokens were used.

1. Include script
<script src="token/token.js"></script>
2. Create token & redirect
Example: from a landing page
<script>
  // Create a token and redirect to a protected page
  Token.create({
    target: "secret.html",      // target page
    extraQuery: "from=landing"  // optional extra query
  });
</script>
3. Validate token on target page
<script src="token/token.js"></script>
<script>
  // Optional: hide content by default in CSS
  // body { display: none; }

  if (Token.validate()) {
    document.body.style.display = "block";
  } else {
    // If invalid or already used, send user back
    window.location.href = "index.html";
  }
</script>

For more details and the real implementation, check the source file in the GitHub repository.
